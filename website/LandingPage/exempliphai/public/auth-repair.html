Auth Repair standalone

Use this page if your browser profile gets stuck signed out / “Verifying…” / “Loading…” after a successful Firebase phone verification.

Important: close all other exempliph.ai tabs (and any embedded views) before running. Deleting IndexedDB while another tab has Firebase Auth open can cause the SDK to wedge.

  • Removes Firebase Auth localStorage keys (firebase:authUser:*)
  • Removes Exempliphai shadow auth record (EXEMPLIPHAI_FIREBASE_AUTH_SHADOW)
  • Deletes Firebase Auth IndexedDB database (firebaseLocalStorageDb)
